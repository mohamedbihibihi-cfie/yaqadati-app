# yaqadati / يقظتي

نموذج أولي React + Vite + Tailwind CSS لأنشطة يقظة قصيرة موجّهة لمعلمي المرحلة الابتدائية، بواجهة عربية RTL جاهزة للعرض على شاشة الصف.

## التشغيل المحلي

1. ثبّت الحزم:

   ```bash
   npm install
   ```

2. شغّل خادم التطوير:

   ```bash
   npm run dev
   ```

3. افتح الرابط الذي يظهر في الطرفية، غالبا:

   ```text
   http://localhost:5173
   ```

## الأوامر

```bash
npm run dev
npm run build
npm run preview
```
